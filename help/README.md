# 帮助文档



## 通用

-   偏好设置
  - 开机时启动
  - 提醒小贴士
  - 显示绘图轨迹
    触摸模块、鼠标模块进行鼠标右键绘图 [^mouseright_draw] 或触摸板绘图 [^touchpad_draw] 时显示光标的轨迹。
  - 语言
  - 同意反馈崩溃信息


- 规则设置
  - 导出规则
  - 导入规则
  - 恢复默认
